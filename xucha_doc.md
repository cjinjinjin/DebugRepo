Introduction1.1 Purpose of This DocumentThis document provides comprehensive guidance on deploying Large Language Models (LLMs) using the DLIS (Deep Learning Inference Service) OaaS (Optimization as a Service) Template. It is designed for engineers and data scientists who need to:Deploy optimized LLM models to production environmentsLeverage DLIS infrastructure for model servingUtilize the OaaS template for streamlined deployment workflowsUnderstand best practices for multi-modal model deployment1.2 What is OaaS?OaaS (Optimization as a Service) is a comprehensive framework for optimizing and deploying Large Language Models on DLIS infrastructure. It provides:Automated Model Optimization: Streamlined workflows for model quantization, compilation, and optimizationMultiple Backend Support: Support for both vLLM and TensorRT-LLM inference enginesProduction-Ready Templates: Pre-configured templates for rapid deploymentPerformance Tuning: Built-in optimization strategies for latency and throughputBatch Processing: Efficient batch inference for high-throughput scenarios📘 Official Documentation: For comprehensive information about optimizing LLM models with the DLIS LLM engine, including detailed configuration options, best practices, and advanced features, please refer to:Optimizing a LLM model with DLIS LLM engineThis official documentation contains extensive information on model optimization techniques, performance tuning strategies, and troubleshooting guidance that complements this deployment guide.2.
